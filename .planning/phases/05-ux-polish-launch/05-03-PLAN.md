## Tasks

### 1. First-Login Onboarding Wizard

**File**: `client/src/components/onboarding/OnboardingWizard.tsx` (new)

Triggered on first login (check `onboarding.getStatus` tRPC call). Full-screen modal wizard:

**Step 1: Welcome**
```
"Welcome to Bottleneck Bots!"
"Let's get your first AI agent running in 3 steps."
[Continue] button
```

**Step 2: Connect GoHighLevel**
```
"Connect your GHL account to unlock 48 automation functions."
[Connect GoHighLevel] button -> initiates OAuth flow
[Skip for now] link -> allows proceeding without GHL
Status indicator: "Connected to [Location Name]" when done
```

**Step 3: Train Your Agent (Optional)**
```
"Upload a SOP or process document so your agent knows your workflows."
[Upload Document] button -> opens Training page uploader
[Use Default Training] button -> skips with built-in knowledge
Brief explanation: "You can always add more training later."
```

**Step 4: Run Your First Task**
```
"Try running your first agent task!"
Pre-filled task input: "Search for contacts tagged 'new-lead' in GoHighLevel"
[Run Task] button -> navigates to Agent Dashboard with pre-filled task
Or: [Browse Templates] -> shows task templates
```

**Completion**:
```
"You're all set! Your agent is ready to work."
Quick links: Agent Dashboard, Templates, Settings
[Go to Dashboard] button
```

Mark onboarding complete via `onboarding.complete` tRPC mutation.

**How-To Articles** (collapsible sections):

- How to create your first agent task
- How to use task templates
- How to train your agent with SOPs
