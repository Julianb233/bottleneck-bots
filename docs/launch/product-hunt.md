## Maker Comment (first comment on the listing)

Hey Product Hunt! 👋

I'm Julian, founder of AI Acrobatics — a one-man AI automation agency.

I built Bottleneck Bots because I was drowning in GoHighLevel setup work. Every new client meant hours of funnel building, workflow configuration, campaign setup, and contact management. I hired VAs, but the bottleneck just shifted.

So I built AI agents that do it instead.

**How it works:**

1. Tell the agent what you need ("Set up a lead capture funnel for my real estate client")
2. The AI plans the task, breaks it into steps, and executes them
3. A real browser agent logs into GHL and does the work — you can watch it live
4. Results are verified and you get a completion report

**What makes it different:**

- Not a template library — actual AI agents that navigate and operate inside GHL
- 48 pre-trained automation functions covering the full GHL feature set
- Multi-agent swarms for complex, multi-step tasks
- White-label architecture so agencies can resell it
- Scheduled automations that run on their own

**Who it's for:**

- Digital marketing agencies using GoHighLevel
- GHL SaaS operators managing multiple sub-accounts
- Freelancers who manage client GHL accounts

**Tech stack for the curious:**

React 19 + tRPC + Supabase + BrowserBase/Stagehand + Claude/Gemini AI + Stripe + BullMQ

I'd love your feedback. This is v1 and I'm iterating fast based on what users need.

AMA in the comments! 🚀
