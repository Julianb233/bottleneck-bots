## Product Summary

Bottleneck Bots is an AI-powered automation platform built for GoHighLevel (GHL) agencies. It deploys intelligent browser agents that log into GHL and execute tasks autonomously — building funnels, managing campaigns, scheduling appointments, and handling client operations across multiple sub-accounts.

The platform includes 48 pre-trained automation functions covering the full GHL feature set, real-time browser session monitoring, scheduled task automation, multi-agent coordination, and a white-label multi-tenant architecture for agency resale.

---

## Key Facts

- **48 pre-trained GHL automation functions** across contacts, calendars, funnels, workflows, campaigns, opportunities, and more
- **Real-time browser agents** powered by BrowserBase and Stagehand — users can watch AI navigate GHL live
- **Multi-agent swarms** that coordinate on complex, multi-step tasks
- **White-label architecture** — agencies can resell under their own brand
- **Scheduled automation** — recurring tasks run on cron schedules without intervention
- **Built by a practitioner** — founder runs a 22-agent AI automation agency

## Technical Architecture

- **Frontend**: React 19, Tailwind CSS 4, shadcn/ui
- **Backend**: Node.js, Express, tRPC 11
- **Database**: Supabase PostgreSQL via Drizzle ORM
- **AI**: Anthropic Claude, Google Gemini, OpenAI
- **Browser Automation**: BrowserBase SDK + Stagehand
- **Payments**: Stripe
- **Background Jobs**: BullMQ + Redis
- **Monitoring**: Sentry
